# InvoiceForge Project

## Core Value

InvoiceForge delivers professional invoice management within WordPress, enabling businesses to create, manage, and deliver invoices seamlessly. It combines WordPress integration with modern PHP practices to provide a secure, extensible solution for small to medium businesses.

## Constraints

### Technical Constraints
- **WordPress Version:** 6.0+ (current APIs and features)
- **PHP Version:** 8.1+ (modern features, strict types)
- **Architecture:** PSR-4 autoloading, dependency injection
- **Security:** WordPress security guidelines, input validation, prepared statements

### Business Constraints
- **Domain:** Invoice management (financial data handling)
- **Integration:** Native WordPress plugin (no external dependencies beyond libraries)
- **Standards:** WordPress coding standards, PHP-FIG recommendations

### Scope Constraints
- **v1 Focus:** Core invoice creation, PDF generation, basic client management
- **Deferred:** Email integration, advanced tax calculations, multi-currency
- **Quality:** Production-ready code with comprehensive testing

## Success Criteria

- Plugin successfully activates/deactivates in WordPress
- Users can create and manage invoices through WordPress admin
- PDF invoices generate correctly with professional formatting
- All security best practices implemented
- Code follows WordPress and PHP standards</content>
