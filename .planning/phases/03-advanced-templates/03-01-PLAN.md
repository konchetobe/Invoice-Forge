From src/Admin/Assets.php (script enqueueing pattern):
```php
public function enqueueScripts(string $hook_suffix): void {
    if (!$this->isInvoiceForgePage($hook_suffix)) { return; }
    if ($this->isSettingsPage($hook_suffix)) { wp_enqueue_media(); }
    wp_enqueue_script(self::JS_HANDLE, ..., ['jquery'], INVOICEFORGE_VERSION, true);
}
private function isSettingsPage(string $hook_suffix): bool {
    return str_contains($hook_suffix, 'invoiceforge-settings');
}
```
